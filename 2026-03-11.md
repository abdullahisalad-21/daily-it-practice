4. 🧩 Steps Completed
4.1 Email Protocols (POP3, IMAP, SMTP)
POP3 downloads and removes mail from server
IMAP syncs mail across devices
SMTP handles outgoing mail
IMAP is the modern standard for multi-device environments
4.2 SSL vs TLS in Email Security
SSL is deprecated
TLS is the secure modern protocol
STARTTLS upgrades insecure connections
Best practice: SMTP 587 + TLS, IMAP 993
4.3 Spam Mitigation (SPF, DKIM, DMARC)
SPF: Defines allowed sending servers
DKIM: Cryptographic signature for authenticity
DMARC: Policy + reporting based on SPF/DKIM
Critical for preventing spoofing and phishing
4.4 File Services (NFS, SMB, Samba, NAS)
NFS: Linux/Unix protocol, fast, used in virtualization
SMB: Windows protocol, universal, supports ACLs
Samba: Linux implementation of SMB
NAS: Hardware device using NFS/SMB for storage

Kernel requirements

4.7 Mobile Synchronization (Android & iOS)
Android backs up app data, SMS, call history, settings, photos
iOS backs up messages, app data, photos, settings, contacts
Both restore during new device setup
4.8 Microsoft 365 & Google Workspace Sync
Microsoft 365 uses Outlook Mobile, Office Mobile, OneDrive

Account‑based sync ensures consistent data across devices
