4.2 Windows Disk & Filesystem Concepts
FAT32 vs NTFS
- FAT32 ‚Üí older, simple, widely compatible
- NTFS ‚Üí modern, secure, supports permissions, journaling
MBR vs GPT
- MBR ‚Üí older, 4 primary partitions, 2TB limit
- GPT ‚Üí modern, supports large disks, required for UEFI
Disk Partitioning Workflow
- Select disk (Disk 0)
- Create primary partition
- Choose filesystem (NTFS or FAT32)
- Format partition
- Assign drive letter
- Mount/unmount volume
Formatting
- NTFS for Windows system drives
- FAT32 for compatibility (USB, external devices)
