# 1. Overview ⭐
Today’s session focused on deep Active Directory architecture concepts from Module 4, including domain-level and forest-level administrative groups, multi-domain forest design, group scope, security principals, EFS, and AD functional levels. We also analyzed your own AD environment screenshot to determine your account’s group membership.

# 2. Objectives 🎯
- Understand the differences between Domain Admins, Enterprise Admins, Domain Users, Domain Computers, and Domain Controllers.
- Learn how multi-domain forests operate and how to design group memberships using AGUDLP/AGUDLP.
- Study group scopes (Domain Local, Global, Universal) and their correct usage.
- Understand security principals, SIDs, access tokens, ACLs, and DACLs.
- Learn the technical features of EFS.
- Study Active Directory functional levels (2012 R2 → 2025).
- Analyze your own AD user account and determine its permissions.
